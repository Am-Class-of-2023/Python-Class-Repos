To enable API functionality:

>Open Command Prompt
>Type "pip install requests" and wait
>If successful, close this program and run it again.
>Otherwise, I don't know how to help you.

IMPORTANT: To end the game properly, press ESC.
Otherwise, your high score won't be written to the leaderboard.
Additionally, the Python file may have to be in its own folder in order to write the highscore file.

Use LEFT and RIGHT arrow keys to turn.
If you go out of bounds, you will loop around the screen.

Eating food will give you a boost.
Using a boost will boost you forward a short distance.
While boosting, you are invulnerable and can pass through walls.
You can use a boost with the UP arrow key.
